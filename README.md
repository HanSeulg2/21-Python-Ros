# Robot Operating System - 미로찾기, 로봇청소기
- Language : Python
- OS : Ubuntu Linux 16.04
- Tools : PyCharm
- Open Source : Turtlebot3/gazebo
- 형성관리 : Smartgit

## 미로찾기
- BFS로 미로의 모든 경로를 탐색한다.
- 미로를 탈출한 이후 Back Tracking으로 최단 경로로 출발지점으로 복귀한다.

## 로봇청소기
- 좌수법으로 정해진 장소의 범위를 파악한다.
- 이후 내부의 모든 장소를 방문하여 청소를 진행한다.
